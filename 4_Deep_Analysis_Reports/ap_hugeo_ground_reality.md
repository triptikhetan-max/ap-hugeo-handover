| Section | Format | Questions | Time | Weight |
|---|---|---|---|---|
| **Section I** | Multiple Choice (MCQ) | 60 questions | 60 minutes | 50% |
| **Section II** | Free Response (FRQ) | 3 questions (7 pts each) | 75 minutes | 50% |

- No penalty for wrong answers — answer everything

## 2. MCQ SECTION — DETAILED BREAKDOWN

### Format
- **60 questions in 60 minutes** (1 minute per question average)
- **ALL questions have 5 answer choices (A through E)** — never 4
- Both **individual questions** and **set-based questions**
- **5-8 stimulus sets**, each with **2-3 questions** sharing the same stimulus

### Stimulus Material
- **30-40% of MCQs reference stimulus material** (~18-24 out of 60)
- Stimulus types: maps, tables, charts, graphs, images, infographics, landscapes
- **Roughly evenly divided between quantitative and qualitative sources**
  - Quantitative: data tables, graphs, charts, population pyramids
  - Qualitative: thematic maps, photographs, images, infographics, landscapes

### 8 MCQ Question Types (per Barron's)

| Type | What it tests | Speed |
|---|---|---|
| 1. Stimulus-Response | Interpret a map/graph/image then answer | 60-90 seconds |
| 2. Definitional | Define a concept, model, or theory | 20-30 seconds |
| 3. Cause-and-Effect | Given a cause, identify the effect | Medium |
| 4. Generalization | From specific case → general principle | Medium |
| 5. Solution to Problem | Define + apply concept to solve | Medium-Slow |
| 6. Hypothetical Situation | Apply knowledge to hypothetical scenario | Medium |
| 7. Chronological | Order/date notable events or processes | Fast |
| 8. Compare/Contrast | Compare two events, models, or concepts | Medium |

**From 2020 onward, stimulus-based questions make up close to HALF of all MCQs.**

## 3. FRQ SECTION — DETAILED BREAKDOWN

| FRQ | Stimulus | Points | Focus |
|---|---|---|---|
| Q1 | **None** (text scenario only) | 7 (parts A-G, 1pt each) | Define → Describe → Explain |
| Q2 | **One** stimulus (map/chart/image/data) | 7 (parts A-G, 1pt each) | Interpret → Apply → Analyze |
| Q3 | **Two** stimuli (map + data, etc.) | 7 (parts A-G, 1pt each) | Compare → Synthesize → Evaluate |

- ~25 minutes per question
- At least 2 of 3 FRQs test spatial analysis skills
- FRQs progress from definition → application → analysis within each question
- Task verbs: **Define, Describe, Identify, Compare, Explain**

### Recent FRQ Topics Tested (from released exams)

**2024 Set 1:**
- Q1: Food availability, agriculture, global trade (carrying capacity, sustainability)
- Q2: Ethnic neighborhoods map — Los Angeles County (cultural patterns, migration)
- Q3: (Stimulus-based — political/economic geography)

**2023 Set 1:**
- Q1: Rate of Natural Increase (RNI), total fertility rate, urban vs rural demographics, ethnonationalism, pronatalist policies, unitary vs federal governance
- Topics spanned: Units 2 + 4

**2023 Set 2:**
- Q1: Territoriality and sovereignty — UK Brexit, Scottish independence, redistricting
- Q2: Human Development Index (HDI), UN Sustainable Development Goals, women's education
- Topics spanned: Units 4 + 7

**Pattern:** FRQs often cross multiple units within a single question.

## 7. KEY MODELS & THEORIES — Must-Know for the Exam

### Unit 1: Thinking Geographically
| Model/Concept | What it is |
|---|---|
| Latitude & Longitude | Coordinate system for mapping Earth's surface |
| GIS | Geographic Information Systems — analyzing data using map layers |
| Thematic Maps | Isoline, dot, proportional symbol, choropleth maps |

### Unit 2: Population & Migration
| Model/Theory | What it is |
|---|---|
| **Demographic Transition Model (DTM)** | 5 stages of population change based on birth/death rates |
| **Epidemiologic Transition Model** | Disease patterns linked to each DTM stage |
| **Malthusian Theory** | Population grows exponentially, food grows linearly → crisis |
| **Boserup's Theory** | Revision of Malthus — innovation increases food production with population |
| **Ravenstein's Laws of Migration** | Common migrant characteristics — short distance, step migration, young males |
| **Zelinsky Model of Migration Transition** | Migration types by DTM stage — rural→urban in stage 2, urban→suburban later |
| **Population Pyramids** | Age-sex distribution diagrams |

### Unit 3: Cultural Patterns
| Model/Theory | What it is |
|---|---|
| **Diffusion S-Curve** | Cultural adoption: slow start → rapid growth → plateau |
| **Language Tree** | Genealogical classification of language families |
| Types of Diffusion | Expansion (contagious, hierarchical, stimulus) vs. relocation diffusion |

### Unit 4: Political Geography
| Model/Theory | What it is |
|---|---|
| **Organic Theory (Ratzel)** | States as living organisms needing expansion |
| **Heartland Theory (Mackinder)** | Control of Eurasian interior = control of world |
| **Rimland Theory (Spykman)** | Control of coastal rim of Eurasia = control of world |
| **Domino Theory** | Communist expansion spreads to neighboring countries |
| **Core-Periphery Model** | Wealthy core nations vs. dependent periphery |
| Centripetal vs. Centrifugal Forces | Forces that unite vs. divide a state |
| Supranationalism | States cooperating (EU, UN, NATO) |

### Unit 5: Agriculture
| Model/Theory | What it is |
|---|---|
| **Von Thunen's Model** | Agricultural land use organized by distance from market (perishability + transport cost) |
| Agricultural Revolutions | First (Neolithic), Second (18th c.), Third (Green Revolution) |

### Unit 6: Cities & Urban
| Model/Theory | What it is |
|---|---|
| **Burgess Concentric Zone Model** | City as concentric rings from CBD outward |
| **Hoyt Sector Model** | Land use in wedge-shaped sectors along transport lines |
| **Harris-Ullman Multiple Nuclei Model** | Multiple business centers, not single CBD |
| **Galactic/Peripheral City Model** | Edge cities around beltways |
| **Latin American City Model (Griffin-Ford)** | Spine/sector model with elite residential sector |
| **Christaller's Central Place Theory** | Hexagonal service areas — larger cities farther apart |
| **Rank-Size Rule (Zipf)** | nth city = 1/n × population of largest city |
| **Primate City** | One city disproportionately larger than all others |
| **Gravity Model** | Interaction between places based on population × distance |
| **Bid-Rent Theory** | Land cost decreases with distance from CBD |
| **Borchert's Urban Evolution** | 5 stages of American urbanization by transport technology |

### Unit 7: Industrial & Economic Development
| Model/Theory | What it is |
|---|---|
| **Rostow's Stages of Growth** | 5 stages from traditional → mass consumption |
| **Wallerstein's World Systems Theory** | Core → semi-periphery → periphery global structure |
| **Dependency Theory** | Underdevelopment caused by colonial exploitation |
| **Weber's Least Cost Theory** | Factory location based on minimizing transport + labor costs |
| **HDI (Human Development Index)** | Composite measure: life expectancy, education, income |
| **GII (Gender Inequality Index)** | Measures gender-based disadvantage |

## 9. WHAT A "GOOD" AP PRACTICE TEST SHOULD LOOK LIKE

Based on all research above, a practice test mimicking the real AP exam should have:

| Dimension | Target |
|---|---|
| Total MCQs | 60 per practice test |
| Choices per question | **5 (A-E), always** |
| Stimulus coverage | **30-40%** of MCQs (~18-24 out of 60) |
| Stimulus format | **5-8 stimulus sets of 2-3 questions each** |
| Stimulus types | Maps (~30%), Tables/Charts (~20%), Passages (~15%), Images (~15%), Diagrams (~10%), None (~45%) |
| Quantitative vs Qualitative | Roughly 50/50 split among stimulus questions |
| Answer position balance | Each position (A-E) correct ~20% of the time |
| Unit distribution | U1: 8-10%, U2-U7: 12-17% each |
| Skill distribution | S1: 25-36%, S2: 16-25%, S3/S4/S5: 13-20% each |
| Topic breadth | 30-40+ distinct topics across 60 questions |
| Question types | Mix of definitional, stimulus-response, cause-effect, compare-contrast, hypothetical, problem-solving |
| Difficulty | Progressive mix (some easy, mostly medium, some hard) |
| Time pressure | 1 minute per question average |
| FRQs (if included) | 3 FRQs: no stimulus, 1 stimulus, 2 stimuli, 7 parts each |
