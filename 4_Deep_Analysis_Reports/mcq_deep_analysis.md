## 9. QUESTION TYPE EXAMPLES

### Type 1: Definition/Recall (6.3% of all Qs)
> "When two groups who speak different languages interact, they often make up a simplified language combination. This is called a(n)"
> - Choices: dialect, **pidgin** ✓, pig tongue, Creole, accent

### Type 2: "Which of the following" (27.9%)
> "Which of the following best explains the level of connectivity between London and New York City?"
> - Longer choices describing different geographic concepts

### Type 3: EXCEPT/NOT (9.1%)
> "All of the following characterize the Green Revolution EXCEPT:"
> - 4 true statements + 1 false (the answer)
> - Princeton Review signature pattern

### Type 4: Example-identification (6.7%)
> "The forced expulsion and killing of specific cultural groups in the former Yugoslavia during the 1990s is an example of"
> - Choices: insurgency, devolution, chain migration, **ethnic cleansing** ✓, Balkanization

### Type 5: Stimulus-based (16.4%)
> "Based on the map/graph above, which of the following countries is the most probable location?"
> - Requires interpreting visual data before answering

### Type 6: Cause/Effect (3.6%)
> "China's one-child policy has led to which of the following demographic challenges?"
> - Tests understanding of geographic processes and consequences

### Type 7: Compare/Contrast (4.6%)
> "How do patterns of urbanization in Europe differ from those in North America?"
> - Tests ability to compare geographic phenomena across regions

### Type 8: Application/Scenario (2.8%)
> "Given Bangladesh's data, its position in the demographic transition model is most likely..."
> - Tests ability to apply models to real-world data

## 10. WHAT THIS MEANS FOR BUILDING QUESTIONS

### The Formula for a Good AP HuGeo MCQ:

1. **Format:** Always 5 choices (A-E), never 4
2. **Stem length:** Target 80-120 characters (median ~90)
3. **Choice length:** Mix of short (25%) and medium (30%) with some long (15%) — avg ~35 chars
4. **Answer balance:** Each position 18-22% (no position should be <15% or >25%)
5. **Stimulus:** 30-40% of practice test questions should reference a map/graph/table/image
6. **Stimulus grouping:** 2-3 questions per stimulus (never 1:1)

### Question Type Mix (per 60-question practice test):

| Type | Target Count | Target % |
|---|---|---|
| Stimulus-based (interpret data) | 18-24 | 30-40% |
| "Which of the following" (general) | 8-12 | 13-20% |
| EXCEPT/NOT | 3-6 | 5-10% |
| Definition/Recall | 4-6 | 7-10% |
| Example-identification | 3-5 | 5-8% |
| Cause/Effect | 3-5 | 5-8% |
| Compare/Contrast | 2-4 | 3-7% |
| Application/Scenario | 2-4 | 3-7% |
| Reasoning/Explanation | 1-3 | 2-5% |

### Key Concepts That Must Be Covered:

**High frequency (should appear in every practice test):**
- DTM, RNI/CBR/CDR, Population pyramids
- Suburbanization, Gentrification, Edge cities
- Cultural diffusion types
- Von Thunen model
- Green Revolution

**Medium frequency (should appear in most tests):**
- Gravity model, Centripetal/centrifugal forces
- Urban models (Concentric Zone, Sector, Multiple Nuclei)
- Malthus, Rostow, Wallerstein
- Gerrymandering, Lingua franca
- Central Place Theory, Rank-size rule

**Should be tested but often missed:**
- Ravenstein's Laws, Zelinsky, Boserup
- Bid-rent theory
- Carrying capacity
- Dependency theory
- Heartland/Rimland theory
