**Enhancement layer (good to have):** Side-by-side comparison that makes the distinction unmistakable:

| | Centripetal / Centrifugal | Push / Pull |
|---|---|---|
| **Acts on** | A STATE (political entity) | An INDIVIDUAL (migrant) |
| **Framework** | Forces that UNITE (centripetal) or DIVIDE (centrifugal) a country | Factors that REPEL from origin (push) or ATTRACT to destination (pull) |
| **CED Unit** | Unit 4 (Political) | Unit 2 (Population & Migration) |
| **Example** | Shared national language = centripetal force holding Spain together | Job opportunities in the US = pull factor for Mexican migrant |
| **NOT this** | "People are pushed to leave" -- WRONG framework for centripetal/centrifugal | "The country is pulled apart" -- WRONG framework for push-pull |

This comparison directly addresses Branson's FRQ 3 error: he used push-pull vocabulary when answering a centripetal/centrifugal question.
