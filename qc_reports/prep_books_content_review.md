## 3. EXAM STRATEGY — What Both Books Teach

### Barron's MCQ Strategy:
1. **8 question types:** Stimulus-Response, Definitional, Cause-Effect, Generalization, Solution to Problem, Hypothetical, Chronological, Compare/Contrast
2. **Key advice:** Read all 5 choices before answering, eliminate obviously wrong answers first
3. **Negative questions** (EXCEPT/NOT) are rare on the real exam
4. **Stimulus questions** require interpreting maps, graphs, tables before answering
5. **Guess if unsure** — no penalty for wrong answers

### Princeton Review MCQ Strategy:
1. **5 question types:** Map, Graph/Table, Theory, Comparison, EXCEPT
2. **4-step process:** Read & Rephrase → When/Who/Where/What? → Process of Elimination → Guess & Go
3. **POE (Process of Elimination)** is the core strategy — mark true/false for each choice
4. **For EXCEPT questions:** Check-mark each TRUE answer, find the FALSE one
5. **Letter of the Day:** If stuck, always guess the same letter for consistency

### Barron's FRQ Strategy:
1. Answer as accurately, clearly, and succinctly as possible
2. Readers look for correct responses, not essay quality
3. Include relevant real-world examples and current events
4. Study by comparing/contrasting related terms
5. Practice with rubrics and sample responses

### Princeton Review FRQ Strategy:
1. **Task verbs matter:** Define, Describe, Explain, Identify, Compare each require different depth
2. Dedicated 9,942-word chapter on FRQ strategy
3. Practice interpreting stimuli before writing
4. Scaffold answers: definition → application → analysis

## 5. EXAM TIPS — What Gets Tested Most

### Barron's Exam Tips (key warnings):

**Unit 2:** "Population pyramids are tested extensively. Know how to read and interpret them."

**Unit 4:** "Gerrymandering has appeared in every recent exam. Know different types: packing vs. cracking."

**Unit 5:** "Von Thunen's model is a favorite exam topic. Know its assumptions AND its real-world limitations."

**Unit 6:** "Urban models (Burgess, Hoyt, Harris-Ullman) are ALWAYS tested. Be able to compare all three."

**Unit 7:** "Deindustrialization is a popular topic for FRQs. Know US history + global patterns."
"Rostow's model has appeared as FRQ — discuss assumptions, predictions, strengths, weaknesses."
"Globalization questions are extremely likely. Know uneven economic benefits."

### Princeton Review Exam Tips:

Every chapter has CED reference boxes linking content to specific College Board topics (e.g., "CED 2.5: The Demographic Transition Model"). This makes it clear which topics are testable.

Princeton emphasizes that students should be able to:
- **Apply models** to real-world situations (not just define them)
- **Analyze at multiple scales** (local, national, global)
- **Interpret visual data** (maps, graphs, pyramids)
- **Make connections across units** (FRQs always test cross-unit knowledge)

## 6. WHAT THIS MEANS FOR BUILDING OUR COURSES

### Content Coverage Requirements:

| Unit | Min Key Terms | Min Models | Min Figures | Must-Cover Topics |
|---|---|---|---|---|
| U1 | 80+ | 3 (Gravity, Distance Decay, Diffusion types) | 15+ (map types, projections) | Map types, GIS, Scale, Regions, Diffusion |
| U2 | 90+ | 6 (DTM, Epi Trans, Malthus, Boserup, Ravenstein, Zelinsky) | 25+ (pyramids, maps) | DTM stages, RNI/CBR/CDR/TFR, Push-pull, Migration types |
| U3 | 120+ | 3 (Diffusion S-curve, Language tree, Cultural landscape) | 15+ (religion maps, language maps) | Diffusion types, Religion, Language, Acculturation |
| U4 | 100+ | 4 (Heartland, Rimland, Organic, Domino) | 8+ (boundary maps) | Boundaries, Gerrymandering, Devolution, Centripetal/Centrifugal |
| U5 | 80+ | 3 (Von Thunen, Agricultural revolutions, Bid-rent) | 8+ (farming type maps) | Von Thunen, Green Revolution, Subsistence vs. commercial |
| U6 | 80+ | 8+ (Burgess, Hoyt, Nuclei, Galactic, Latin Am, Central Place, Rank-size, Bid-rent) | 10+ (city model diagrams) | Urban models (compare!), Gentrification, Sprawl, Edge cities |
| U7 | 100+ | 5 (Rostow, Wallerstein, Weber, Dependency, Core-Periphery) | 10+ (development maps) | Rostow stages, HDI, Globalization, Maquiladoras |

### Question Style Should Mirror:

1. **~30% "Which of the following"** stems (Barron's style)
2. **~10% EXCEPT/NOT** (Princeton style — important to include)
3. **~15% definition/recall** ("known as", "refers to")
4. **~10% example-identification** ("this is an example of")
5. **~15% stimulus-based** interpretation
6. **~10% cause/effect**
7. **~10% application/scenario**

### Key Teaching Gaps to Watch For:

1. **International urban models** — Princeton covers Latin American, SE Asian, African, Islamic city models; Barron's doesn't. The AP exam tests these.
2. **Agricultural specifics** — Both books under-test U5. Include more farming types, commodity chains, land survey systems.
3. **Scale Analysis** — Both books mention it but don't drill it enough. Students need practice analyzing patterns at local vs. national vs. global scales.
4. **Cross-unit connections** — FRQs always test across units. Practice questions should too.
