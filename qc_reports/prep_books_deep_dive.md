## 4. QUESTION STEM PATTERNS

### Most Common Question Types (ranked by frequency):

| Pattern | Barron's | Princeton Review | Notes |
|---|---|---|---|
| **"Which of the following..."** | **#1** (75% in PT2!) | **#1** (21%) | Dominant framing in both |
| **EXCEPT / NOT** (inverted) | Rare | **#2** (~19%) | Princeton LOVES these |
| **"All of the following EXCEPT"** | Rare | 11.5% | Princeton signature pattern |
| **Definition / Recall** | Moderate | 13% | "This is known as..." |
| **Example-identification** | Moderate | 11% | "This is an example of..." |
| **Cause / Effect** | Some | 8.5% | "What led to..." |
| **Compare / Contrast** | Some | 7.3% | "How does X differ from Y?" |
| **Stimulus interpretation** | ~25-43% | ~19% | "Based on the map above..." |
| **Scenario / Application** | Some | 3.8% | Hypothetical situations |
| **Fill-in-the-blank** | A few | Rare | "_________ maps work well when..." |

### Key Differences Between Publishers:

**Barron's Style:**
- Heavier on "Which of the following" (up to 75% of questions in some tests)
- More stimulus-heavy (especially in later tests: ~43-47%)
- More real-world application questions
- Less use of EXCEPT/NOT pattern
- Tends toward longer, more nuanced stems

**Princeton Review Style:**
- Heavy use of EXCEPT/NOT (19% of all questions — nearly 1 in 5!)
- More definition/recall questions
- More balanced question type mix
- Lower stimulus coverage (~26%)
- More pedagogical answer explanations (3-6 sentences each)
- Explicitly teaches 5 MCQ types: Map, Graph/Table, Theory, Comparison, EXCEPT

## 11. WHAT WE LEARN FOR OUR BUILDS

### Format Requirements (confirmed by all 3 books + College Board):
- ✅ **5 choices (A-E)** — every question, no exceptions
- ✅ **60 MCQs per practice test** — standard format
- ✅ **3 FRQs per test** — 7 parts each, no/1/2 stimuli
- ✅ **30-40% stimulus-based** — grouped in sets of 2-3 questions
- ✅ **Answer positions balanced** — target ~20% each (A through E)

### Question Style Patterns to Replicate:
1. **"Which of the following"** — the dominant stem pattern (20-75% of questions)
2. **EXCEPT/NOT** — Princeton uses these heavily (~19%), Barron's less so
3. **Definition/Recall** — "This is known as..." / "This is referred to as..."
4. **Example-identification** — Real-world scenario → classify the concept
5. **Stimulus interpretation** — "Based on the map/graph/table above..."
6. **Cause/Effect** — "What led to..." / "What is a consequence of..."
7. **Compare/Contrast** — "How does X differ from Y?"
8. **Application/Scenario** — Hypothetical situations requiring concept application

### Stimulus Types to Include:
- Thematic maps (most common)
- Data tables (country statistics, demographics)
- Graphs (bar, line, pie charts)
- Population pyramids
- Photographs/images
- Models/diagrams (urban models, DTM stages)
- Text passages (shorter excerpts)

### Cognitive Level Mix:
- ~15-20% Recall (know the term)
- ~45-55% Comprehension (understand and interpret)
- ~15-25% Application (apply to new situation)
- ~10-15% Analysis (evaluate, compare, synthesize)

### Topic Balance:
- U1 gets fewer questions (8-10%)
- U2-U7 roughly equal (12-17% each)
- U5 (Agriculture) tends to be under-represented in prep books — we should ensure adequate coverage

### Common Pitfalls in Prep Book Questions:
- Answer position skew (some positions overrepresented by 10%+)
- Stimulus coverage below 30% in many tests
- U5 consistently under-tested
- Cognitive level skews toward comprehension over application/analysis
- FRQ topics sometimes repeated across tests (Barron's reuses deindustrialization theme)
